NasaTools Debian repository

Como Usar

Fazer download da chave pública:

sudo bash -c 'wget -q -O - https://apt.sre.nasajon.com.br/public.key | gpg --dearmor | tee /usr/share/keyrings/nasajon-archive-keyring.gpg 2>&1 > /dev/null'

Adicionar o repositório:

  sudo bash -c 'echo "deb [arch=amd64 signed-by=/usr/share/keyrings/nasajon-archive-keyring.gpg] http://apt.sre.nasajon.com.br stable main" | tee /etc/apt/sources.list.d/nasajonsre.list'

Executar:

sudo apt-get update
sudo apt-get install python3-nasatools
